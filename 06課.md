# を: 動作作用對象

ex:
わたしは　しんぶん　を　読みます

# で: 動作進行的地點

ex:
わたし　は　駅で　しんぶん　を　かいます

# 要不要...(邀請問法)

<pre><span class="import">いっしょに（一緒に）</span>　こうべ　へ　行き<span class="import">ませんが</span></pre>

# 做...吧

<pre>ちょっと　休み<span class="import">ましょう</span></pre>

# .... それから ....

<pre>うちで宿題をします　<span class="import">それから</span>　CDを聞きます</pre>
## 接述詞
それから：然後
ときどき：偶爾
